# PhD Portfolio Website

Welcome to the repository for my PhD portfolio website. This site showcases my academic journey, research interests, and projects as I pursue my PhD in Computer Science.

## Overview

This portfolio is built using GitHub Pages and Jekyll, providing a clean, professional presentation of my academic and professional achievements.

## Contents

- `index.md`: The main content of the portfolio website.
- `_config.yml`: Jekyll configuration file.
- `assets/`: Directory containing additional resources like images and PDFs.

## Features

- Responsive design
- Timeline of academic history
- Project showcases
- News and updates section

## Local Development

To run this site locally:

1. Install Jekyll (if not already installed):
   ```
   gem install jekyll bundler
   ```

2. Clone this repository:
   ```
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

3. Install dependencies:
   ```
   bundle install
   ```

4. Run the Jekyll site:
   ```
   bundle exec jekyll serve
   ```

5. Open your browser and go to `http://localhost:4000`

## Customization

To customize this portfolio:

1. Edit `index.md` to update the content.
2. Modify the CSS in the `<style>` tag at the bottom of `index.md` to change the appearance.
3. Update `_config.yml` to change Jekyll settings or theme.

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
